# MACHINE-LEARNING-PROJECT
# Heart Disease Prediction using Machine Learning

## Project Overview
This project focuses on predicting the presence of heart disease using machine learning algorithms. The dataset was preprocessed, important features were selected, and multiple classification models were trained and compared to identify the best-performing model.

## Dataset
- Source: Kaggle
- Dataset: Heart Disease Prediction Dataset

## Objectives
- Load and preprocess the dataset
- Handle missing values and encode categorical features
- Perform feature engineering
- Split the data into training and testing sets
- Train multiple machine learning models
- Compare model performance using evaluation metrics
- Select the best-performing model

## Machine Learning Models Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Random Forest Classifier

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

## Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab

## Results
All three machine learning models were evaluated and compared. The best-performing model was selected based on Accuracy, Precision, Recall, and F1-Score. A confusion matrix was used to visualize the classification performance.

## Conclusion
This project demonstrates a complete machine learning workflow, including data preprocessing, feature engineering, model training, evaluation, and model comparison. The selected model achieved the best overall performance for heart disease prediction.
